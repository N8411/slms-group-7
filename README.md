# EduTrack – Student Learning Management System (SLMS)

## Overview
**EduTrack** is a **Student Learning Management System (SLMS)** designed to manage **student profiles, course records, and user registrations** in an organized and efficient way.

The system provides a **centralized platform** where **admins, lecturers, and students** can interact with academic information such as courses and student data.

This project was developed as part of an academic exercise focusing on system design, documentation, and structured programming practices.

---

## Features

### Admin
- Manage student records
- Manage course information
- Add, edit, delete, and search system data

### Lecturer
- View courses
- Search courses
- Access student information related to their courses

### Student
- Log in to the system
- Search available courses
- View course information
- Log out
